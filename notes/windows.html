Adding Environment Variables to the PATH

The easiest way to do it is via the GUI.

  • Go to your Window's search and look up "Environemt variables".
  • The Environment Variables settings page will pop up.
  • Find the User Environment Variable section and update the PATH.

It should be noted that the deliminator (in the GUI) for the paths in the PATH are new lines. So it is easy to add an entry, but it did feel weird to do...
